# node_crud_api
# Vanilla Nodejs RestFul CRUD API Project with No Framework or Library
> This is vanilla Node.js CRUD API todo list project.

## Routes
```bash
GET      /api/todos
POST     /api/todos
GET      /api/todos/:id
PUT      /api/todos/:id
DELETE   /api/todoss/:id

```

## Installation

```bash
# Install dependencies
npm install

# Run in develpment
npm run dev

# Run in production
npm start
```
